# FinRelief AI Documentation

## Overview
FinRelief AI is an AI-powered debt management platform built using React, FastAPI, SQLAlchemy, SQLite and Google Gemini AI.

## Features
- JWT Authentication
- Profile Management
- Loan Management
- Dashboard
- Financial Health
- Settlement Prediction
- AI Negotiation
- AI Email Generator
- History
- Know Your Rights

## Technology Stack
Frontend: React, Vite, Axios
Backend: FastAPI
Database: SQLite
AI: Google Gemini API
Deployment: Vercel + Render

## Database Entities
- Users
- Loans
- AI_History

## Future Scope
- PostgreSQL
